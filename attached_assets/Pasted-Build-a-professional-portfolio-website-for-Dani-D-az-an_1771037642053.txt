 - Minimal shadows (rely on layering and transparency)                    
 - Hover effects: translateY(-4px) on cards, opacity changes on buttons

 ## ANIMATIONS & INTERACTIONS                                             
 - Smooth scroll (CSS scroll-behavior: smooth or library like Lenis)      
 - Scroll reveal: elements fade in + translateY(30px→0) as they enter     
 viewport                                                                 
 - Staggered grid animations: 0.1s delay between items                    
 - Page transitions: subtle fade (0.2s)                                   
 - Button hover: opacity-80 or background color shift                     
 - Link underlines: width animates from 0 to 100% on hover                
 - Image containers: overflow-hidden, img scales to 1.05 on hover         
                                                                          
 ## RESPONSIVE BREAKPOINTS                                                
 - Mobile: 1 column grids, hamburger menu, text-5xl for hero              
 - Tablet (768px): 2 column grids, desktop nav, text-7xl hero             
 - Desktop (1024px+): 3-4 column grids, full layout, text-8xl hero        
 - Max container width: 1280px with px-6 padding
