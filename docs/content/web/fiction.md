# Fiction

## Section Intro

Authentic stories start with real faces. I've led **60+** casting processes for **feature films and short films**, working with **directors from around the world**. Several of these projects have been **selected and awarded at renowned festivals.** My work brings together established actors and natural talent, with a multicultural, human approach and a strong focus on **street casting.**
